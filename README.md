# Gemini Enterprise Trial Survey

Interactive Typeform-style survey prototype for Audio/Video Services at World Headquarters.

## Live Preview

Visit: **https://williamproctor.github.io/gemini-trial-survey/**

## Features

- One-question-at-a-time Typeform-style UX
- Branching logic for users vs. non-users
- Dark/light mode with system preference detection
- Fully self-contained single HTML file (no dependencies)
- SharePoint List REST API integration (activates in SP context)

## Usage

This is a **prototype** — open `index.html` in any browser to preview. When deployed to SharePoint, it connects to the `GeminiTrialSurveyResults` list automatically.
